## CNN Explainer : Learn more ##

URL: https://poloclub.github.io/cnn-explainer/

Running locally:
1. $ git clone https://github.com/poloclub/cnn-explainer.git
2. $ cdd cnn-explainer
3. $ npm install
4. $ npm run dev
5. Open browser at http://localhost:5000
